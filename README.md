# Treasure Hunt Maze Game - Escape the Enemy! 🏁

## Overview
This is a Pygame-based maze escape game where the player navigates through a randomly generated maze to reach the treasure while avoiding an enemy that patrols the area.

## Features
- Randomly generated maze for unique gameplay.
- Enemy AI using A* pathfinding.
- Smooth player movement using arrow keys.
- Timer and move counter for performance tracking.
- Image-based player, enemy, and treasure elements.

## Requirements
Ensure you have Python installed along with the necessary dependencies.

### Dependencies
Install the required libraries using pip:
```bash
pip install pygame
pip install numpy
pip install flask
```

## How to Run
1. Clone or download the project files.
2. Ensure you have `girl-1.png`, `enemy.png`, and `treasure.png` in the same directory as the script.
3. Run the game script:
```bash
python maze_game.py
```

## Controls
- **Arrow Keys**: Move the player
- **Play With Friends**: one player controls arrow keys and other player can play in same pc by using 'A'- for moving left, 'D'- for moving right, 'W'- for moving up, 'S'- for moving down.
- **Escape**: Exit the game

## How to Win
- Reach the treasure before getting caught by the enemy!
